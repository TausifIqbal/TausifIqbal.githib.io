Sequencing_Metrics_Tools

This repo hosts scripts/tools that are used for sequencing metrics calculation. Please refer to this confluence pages for the list of metrics

https://rochewiki.roche.com/confluence/display/RSS/Sequencing+Metrics+Summarization

Install dependencies and pytests

For metrics, To install dependencies and running scripts / pytest from Github workflows as shown below as example:

python3.10 -m pip install --upgrade pip
python3.10 -m pip install poetry
cd bfx-ngs/metrics
poetry install --no-root
# Run module scripts
poetry run python src/report/summarize_metrics.py [arguments ...]
# Run tests
poetry run pytest

To run mosdepth - fast BAM/CRAM depth calculation

# Download mosdepth
wget https://github.com/brentp/mosdepth/releases/download/v0.3.4/mosdepth
chmod +x mosdepth
# Run mosdepth
# ex.
# ./mosdepth -t 1 --by 'bed_file' -Q 4 'sample_name' 'sample.bam'
./mosdepth -t 1 --by 'GRCh38_major.bed' -Q 4 'NA12878' 'NA12878.bam'

Specific examples

Clustering Strand Bias

run command: python cluster_strand_balance.py --input_file <Clsuter txt file> --output_dir <Output directory> --samplename <Sample Name>
docker command: docker run --rm -v `pwd`:`pwd` navifyplatform-eug-snap-docker.jfrog.io/bfx-sbx-metrics:0.0.8-SNAPSHOT-avc-551-poerty-dockerfile-latest cluster_strand_balance.py --input_file `pwd`/metrics/cfDNA1.run_fastq_1m.umis.demux.fq.txt --output_dir `pwd`/qc_metrics/cluster_strand_bias/cfDNA1 --samplename cfDNA1

Cluster Size Analysis

run command: python calc_clustersize.py --input_files <Cluster text file> --samplenames <Sample Name> --filetypes AA --output_dir <Output directory>
docker command: docker run --rm -v `pwd`:`pwd` navifyplatform-eug-snap-docker.jfrog.io/bfx-sbx-metrics:0.0.8-SNAPSHOT-avc-551-poerty-dockerfile-latest calc_clustersize.py --input_file `pwd`/metrics/cfDNA6.run_fastq_1m.umis.demux.fq.txt --samplenames cfDNA6 --filetype AA --output_dir <Output directory>

UMI distance

run command: python umi_distances.py --input_file <Cluster text file>
docker command: docker run --rm -v `pwd`:`pwd` navifyplatform-eug-snap-docker.jfrog.io/bfx-sbx-metrics:0.0.8-SNAPSHOT-avc-551-poerty-dockerfile-latest umi_distances.py --input_file `pwd`/metrics/cfDNA1.run_fastq_1m.umis.demux.fq.txt --samplename cfDNA1

Base Level Accuracy
