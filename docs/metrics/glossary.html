SampleMetrics

Field Description
sample_2plus_name The sample name.
sid_sequence The SID sequence.
reads_per_sid The number of reads with this SID.
full_length_reads The number of full length reads with this SID.
partial_reads The number of partial length reads with this SID.
reads_filtered The number of reads filtered out with this SID.

AdapterMetrics

Field Description
sid_5p The 5' SID sequence.
umi_5p The 5' UMI sequence.
umi_3p The 3' UMI sequence.
sid_3p The 3' SID sequence.
pct The percent of reads with this combination of barcodes.
cnt The number of reads with this combination of barcodes.

DemuxMetrics

Field Description
total_reads The total number of reads (molecular traces, or mts) of this HDF5 or FASTQ input file.
total_reads_post_filter The total number of reads post filters, e.g. minimum length filter.
pct_filtered_reads The percent of filtered reads. pct_filtered_reads = (1 - total_reads_post_filter/total_reads) x 100.
total_demuxed_reads The number of reads that are successfully demuxed.
total_undemuxed_reads The number of reads that are not assigned to any sample ID (SID). total_undemuxed_reads = total_reads_post_filter - total_demuxed_reads
total_full_length_reads Number of reads with identified 5’, and 3’ SIDs and unique molecular IDs (UMIs).
total_partial_length_reads Number of reads with identified 5’ SID but excludes the full length reads.
pct_demuxed_reads The percent of successfully demuxed reads out of the total reads post filter. Formatted to %.2d. (pct_of_demuxed_reads) = (total_demuxed_reads) / total_reads_post_filter) x 100
pct_index_hopping Percent of the reads whose 5' SIDs disagree with 3' SIDs. Formatted to %.2d. Index_hopping_rate = (number of reads 5’ SID != 3’ SID)/total reads with both SIDs detected x 100
pct_full_length_reads The percent of full length reads out of the total reads post filter. Formatted to %.2d. pct_full_length_reads = total_full_length_reads/total_reads_post_filter x 100
pct_partial_reads The percent of partial length reads out of the total reads post filter. Formatted to %.2d. pct_partial_length_reads = total_partial_length_reads/total_reads_post_filter x 100
